# Resume Grader AI

> **Transform resumes into recruiter-ready profiles with AI-powered evaluation, actionable feedback, and intelligent optimization.**

Resume Grader AI is an intelligent resume evaluation platform designed to help candidates understand how their resumes perform before they reach a recruiter. By combining modern AI capabilities with structured resume analysis, the platform delivers detailed insights, identifies improvement opportunities, and provides personalized recommendations that enhance resume quality and professional impact.

Whether preparing for internships, campus placements, or experienced roles, Resume Grader AI acts as a virtual resume reviewer—helping users refine every submission with confidence.

---

## Why Resume Grader AI?

Recruiters often spend only a few seconds reviewing a resume. Resume Grader AI bridges the gap between candidate potential and recruiter expectations by providing instant, data-driven feedback that empowers users to present their skills more effectively.

The platform focuses on improving resume quality through intelligent analysis rather than simple keyword matching, making every evaluation meaningful and actionable.

---

## Core Features

* **AI-Powered Resume Analysis** for comprehensive document evaluation.
* **Professional Resume Scoring** with clear performance metrics.
* **Section-wise Feedback** covering structure, clarity, and presentation.
* **Skill & Content Assessment** to highlight strengths and opportunities.
* **ATS-Oriented Recommendations** for improved recruiter visibility.
* **Personalized Improvement Suggestions** generated specifically for each resume.
* **Fast & Interactive Experience** with real-time feedback.
* **Modern Responsive Interface** optimized across desktop and mobile devices.

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/yanipatel64/resume-grader.git
```

### Navigate to the Project

```bash
cd resume-grader
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the project root.

```env
VITE_GEMINI_API_KEY=YOUR_API_KEY
```

### Start the Development Server

```bash
npm run dev
```
